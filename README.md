[Cincom Visualworks](http://cincomsmalltalk.com/) Smalltalk Store.

[Visualworks Smalltalk Store Plugin](http://wiki.jenkins-ci.org/display/JENKINS/Visualworks+Smalltalk+Store+Plugin)
